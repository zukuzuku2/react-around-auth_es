# Política de Seguridad

## Versiones Soportadas

Actualmente, solo la última versión de la rama principal (`main`) recibe actualizaciones de seguridad.

| Versión | Soportada |
| ------- | --------- |
| 1.0.x   | ✅         |
| < 1.0   | ❌         |

## Reportando una Vulnerabilidad

Si encuentras una vulnerabilidad de seguridad, por favor **no abras un reporte público**. En su lugar, puedes contactar al mantenedor del repositorio directamente a través de GitHub o mediante un mensaje privado.

Se revisarán todos los reportes y se intentará dar una solución en el menor tiempo posible. ¡Gracias por ayudar a mantener este proyecto seguro!
